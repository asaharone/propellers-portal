# Пропеллеры · Foundation Portal

Приватный портал по проекту бренда пропеллеров для дронов.

- **Доступ только по паролю.** Статическая страница (этот репозиторий) не содержит никаких чувствительных данных — только экран входа и код интерфейса.
- Все документы, ответы и комментарии хранятся в защищённом бэкенде (Supabase Edge Function + Postgres с RLS) и загружаются **только после ввода пароля**.
- Комментарии, ответы на вопросы и статусы задач **общие**: их видят все, у кого есть пароль.

## Как это устроено

```
GitHub Pages (этот репозиторий) ──►  index.html: экран пароля + интерфейс
                                          │  (ничего секретного в исходнике)
                                          ▼
                         Supabase Edge Function «portal»
                         · проверяет пароль (серверный салт+хэш)
                         · выдаёт подписанный токен сессии (30 дней)
                         · отдаёт контент и читает/пишет комментарии
                                          │
                                          ▼
                         Postgres (RLS закрыт; доступ только функции)
```

## Локальный запуск

```bash
python3 -m http.server 8765 --directory .
# открыть http://localhost:8765
```

Бэкенд — отдельный проект Supabase (`propellers-portal`). Исходник функции и контент-бандл хранятся вне публичного репозитория.
